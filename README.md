# 🛡️ PhishGuard: Multi-Layer Phishing Detection System

PhishGuard is a professional cybersecurity tool designed for Security Operations Centers (SOC) to perform deep forensic analysis of URLs and Email content. It leverages weighted heuristics and dynamic metadata analysis to classify threats.

![Dashboard Preview](https://via.placeholder.com/1200x600/0b0f19/e2e8f0?text=PhishGuard+SOC+Dashboard+Preview)

## 🚀 Features

- **🌐 Website Intelligence Analysis**:
  - **HTTPS & SSL Audit**: Validates secure protocol usage.
  - **Heuristic String Search**: Scans for 50+ deceptive keywords used in social engineering.
  - **IP-Based Domain Detection**: Identifies risky direct-IP access attempts.
  - **WHOIS Domain Longevity**: Analyzes registration age to detect "burner" domains.
  - **DOM Forensics**: Scans for forms without actions and external malicious scripts.

- **📧 Advanced Email Forensics**:
  - **Header Spoofing Check**: Detects mismatches between `From` and `Reply-To` fields.
  - **Malicious Attachment Filter**: Flags risky extensions like `.exe`, `.js`, `.scr`, etc.
  - **Payload URL Extraction**: Recursively extracts and scans every link inside the email body.
  - **Text Analytics**: Detects high-pressure keywords ("urgent", "suspended", "verify").

- **📊 Modern SOC Dashboard**:
  - **Glassmorphism UI**: High-end frosted glass interface with dynamic mesh backgrounds.
  - **Real-Time Stats**: Live tracking of scan volume and threat classifications.
  - **Global Audit Logs**: Persistent SQLite-based logging of every forensic action.
  - **Automated PDF Reporting**: Generates secure document reports for SOC filing.

## 🛠️ Tech Stack

- **Backend**: Python 3.x, Flask
- **Database**: SQLite (SQLAlchemy ready)
- **Frontend**: HTML5, Vanilla CSS (Modern Grid/Glassmorphism), Lucide Icons
- **Forensics Libraries**: `python-whois`, `beautifulsoup4`, `reportlab`

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/phishguard.git
   cd phishguard
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Dashboard**:
   ```bash
   python app.py
   ```
   *Access at `http://127.0.0.1:5000/`*

4. **Use the CLI**:
   ```bash
   python cli.py --url https://suspicious-site.com
   ```
